## ✅ ধাপ ১: সাইট খুলুন (১ মিনিট)

2. "Open with Live Server" ক্লিক করুন

**অথবা সরাসরি:**

## ✅ ধাপ ৩: আপনার ছবি যোগ করুন (১ মিনিট)

**`index.html` এ খুঁজুন:**
```html
<img src="https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d?w=500&h=600&fit=crop" />
```

**যেকোনো জায়গা থেকে ছবির লিংক নিন:**
1. **Unsplash থেকে:** https://unsplash.com
   - কোনো ছবি সার্চ করুন
   - ছবিতে রাইট ক্লিক → "Copy Image Link"
   
2. **নিজের ছবি:**
   - Imgur বা অন্য কোথাও আপলোড করুন
   - লিংক কপি করুন

**এভাবে রাখুন:**
```html
<img src="আপনার_নতুন_ছবির_লিংক" />
```

## 🎉 শেষ!

এখন আপনার সাইট প্রস্তুত! 

### এরপর কি করবেন?

**নতুন পেজ যোগ করতে:**
1. সেরা কোনো পেজ খুলুন (যেমন `services.html`)
2. Ctrl+A → Ctrl+C (সব কপি করুন)
3. নতুন ফাইল তৈরি করুন (যেমন `portfolio.html`)
4. Ctrl+V (পেস্ট করুন)
5. নিজের কন্টেন্ট দিন
6. **Header-Footer অটোমেটিক যুক্ত হবে!**

**আরও বিস্তারিত?**
- `CUSTOMIZE.md` খুলুন এবং পড়ুন বাংলায় সম্পূর্ণ গাইড
